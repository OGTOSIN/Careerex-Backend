.forEach()

The .forEach() method does not create a new array with the result of the call back function - it basically logs the result of the condition(s) you set there (Performs the specified action for each element in an array).

.map()

The .map() method create a new array with the result of the call back function - or basically the result of the condition you set for it to return(Calls a defined callback function on each element of an array, and returns an array that contains the results).

.filter()

The .filter() method does not create a new array with the result of the call back function - it basically return the result of the condition you set (Returns the elements of an array that meet the condition specified in a callback function.).
